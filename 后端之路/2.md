# [FastAPI系列]GET的使用

众所周知我们已经学废了FastAPI最基础的操作，接下来我们讲GET的使用。

GET的框架：

```Python
# 导入和初始化，见上一期
@app.get("/") #这里写路径，我写的是根目录
def man(): # Main! What can I say!
  return {"Hello": "world!"} # 这里是你要返回的内容
```

然后，退出Vim，在终端运行：

```Shell
fastapi dev
```

然后，打开浏览器，输入`https://localhost:8000/`，你就可以看到：`{"Hello":"World"}`！

而且，FastAPI还会自动生成API文档，具体的用途下期再说。
