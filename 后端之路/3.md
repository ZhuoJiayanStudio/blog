# [FastAPI系列]带参数的GET

我也不知道怎么引导了，反正我们要返回前端已有的值，这时候我们就需要**带参数的GET。**

示例：用户登录好的欢迎语

```Python
# 第一期提到的初始化
@app.get("/user/{user_name}")
def user_login(user_name: str):
  return {"welcome": f"欢迎{user_name}！"}
```

然后，我们`fastapi dev`，打开浏览器，为了测试我们的代码，我们可以使用FastAPI内置的文档：`https://localhost:8000/docs`，在里面找到`default`下的`GET user/{user_name} User Login`，点击`Try it out`，输入用户名（`user_name`）卓嘉彦，点击`Execute`，就可以看到返回的结果了：`“string”`，奇怪，`docs`居然抽风了，原来是我们输入了中文，没事，我们在`def`后面加上`user_name = unquote(user_name)`，就行了。再来试试：不行，还是`"string"`，导入`unquote`试试：`from urllib.parse import unquote`，成功了：

```
{
  "welcome": "欢迎卓嘉彦！"
}
```
