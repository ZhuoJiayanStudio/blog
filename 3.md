# 如何向长辈证明电脑的辐射无害
首先，我们可以拿出国家标准，哪有产品敢不符合国标的？[GB 8702-2014 电磁环境控制限值](https://www.mee.gov.cn/ywgz/fgbz/bz/bzwb/hxxhj/dcfsbz/201410/t20141022_290449.shtml)。如果长辈认为奸商不遵守国标的话，可以把电脑（这里只针对笔记本电脑）翻过来让他们看3C认证。

而且，中国计量科学研究院也进行过科普：[电磁辐射知多少 中国计量院电磁环境专家带你认识身边的电磁场](https://www.nim.ac.cn/node/267)。长辈不信你，总不能不信中国计量科学研究院吧。
