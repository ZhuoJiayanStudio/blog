## 改造石山

众所周知，Python的print函数可能会影响测试所需时间，而且终端上输出这么多东西你上哪里看分数？而且main函数还容易撞名，因此，我们需要对代码进行改造。

改造结果如下：

### CPU

```python
def is\_sushu(aa) -> bool:
    for i in range(2, int(aa \*\* 0.5) + 1):
        if aa % i == 0:
            return False
    return True

def CPU() -> int:
    n: int = 5000
    theTime: int = time.time()
    for i in range(4, n + 1, 2):
        b: bool = True
        for j in range(2, i + 1):
            if b == True and is\_sushu(j):
                for k in range(2, i + 1):
                    if b == True and is\_sushu(k):
                        if j + k == i:
                            b = False
    print(f"The CPU's score is {int(10000 // (time.time() - theTime))}/10000")
    return 0
```
