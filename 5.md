# 通过Github Model白嫖DeepSeek AI大模型
## 准备工作
你需要：Python解释器、Github账号（如果没有的话，注册一个，参考[评论功能](https://zjy.gitbook.io/zjyblog#ping-lun-gong-neng)，Github打不开的话，参考[Github加速](https://zjy.gitbook.io/zjyblog/1#github-jia-su)）。
### 安装Python解释器
Linux系统一般有预装Python解释器，输入以下命令：
```bash
python3
```
如果能显示出交互页面，就说明你安装了Python解释器。没预装的自己上网查，注意不要用百度，否则你可能学不会。注意，Linux和MacOS系统的Python命令是"python3"

退出Python交互页面的命令是：
```python
quit()
```
如果是Windows系统，输入以下命令打开Microsoft Store安装或进入交互页面，如果进入了交互页面参考上面的方法退出：
```powershell
python
```
然后，我们就装好Python了。

接下来安装依赖，Windows使用这个命令：
```powershell
pip install openai rich -i https://mirrors.aliyun.com/pypi/simple/
```
Linux或MacOS使用这个命令：
```bash
pip3 install openai rich -i https://mirrors.aliyun.com/pypi/simple/
```
然后就好了。
## 获取API
前往[这个网站](https://github.com/settings/tokens)获取免费API，然后随便找个目录新建一个文件，文件名保存为ai.py，输入以下代码：
```python
import sys
import os
from openai import OpenAI
from rich.console import Console
from rich.markdown import Markdown
from rich.live import Live

# ==================== 1. API 配置 ====================
# 优先从环境变量读取，找不到则读取硬编码的值（记得在下面填入你的 ghp_ 密钥）
API_KEY = os.environ.get("GITHUB_TOKEN", "")  # 👈 也可以直接在这里填入 "ghp_你的密钥"
BASE_URL = "https://models.inference.ai.azure.com"
MODEL_NAME = "deepseek/DeepSeek-V3"

client = OpenAI(api_key=API_KEY, base_url=BASE_URL)
console = Console()

def get_input_prompt():
    """获取用户输入：支持管道标准输入、命令行参数和交互式输入"""
    if not sys.stdin.isatty():
        return sys.stdin.read()
    elif len(sys.argv) > 1:
        return " ".join(sys.argv[1:])
    else:
        console.print("[bold yellow]提示: 未检测到输入。请输入你的问题...[/bold yellow]")
        return input("> ")

def main():
    # 安全校验：检查 API_KEY 是否有效
    if not API_KEY or API_KEY.startswith("ghp_你的"):
        console.print("[bold red]错误: 未检测到有效的 API Key。[/bold red]")
        console.print("[yellow]请选择以下一种方法解决：[/yellow]")
        console.print("1. 修改代码第 10 行，把 API_KEY 改为你的真实密钥。")
        console.print("2. 在终端设置环境变量：")
        console.print("   Windows PowerShell: $env:GITHUB_TOKEN=\"你的Token\"")
        console.print("   Linux/macOS: export GITHUB_TOKEN=\"你的Token\"")
        return

    prompt = get_input_prompt()
    if not prompt.strip():
        return

    try:
        response = client.chat.completions.create(
            model=MODEL_NAME,
            messages=[{"role": "user", "content": prompt}],
            stream=True,
            temperature=0.3, # 降低温度，写代码更严谨
        )

        answer_response = ""

        # 使用 Rich 的 Live 组件动态刷新渲染终端界面
        with Live(Markdown(""), console=console, refresh_per_second=15, auto_refresh=False) as live:
            for chunk in response:
                if not chunk.choices:
                    continue

                delta = chunk.choices[0].delta
                content = getattr(delta, "content", None)
                if not content:
                    continue

                # 累加标准回复内容
                answer_response += content

                # 实时更新渲染 Markdown
                live.update(Markdown(answer_response), refresh=True)

    except Exception as e:
        console.print(f"\n[bold red]API 错误:[/bold red] {e}")

if __name__ == "__main__":
    main()
```
注意要填的东西。

Windows用户运行这个命令：
```powershell
python ai.py
```
MacOS/Linux用户运行这个命令：
```bash
python3 ai.py
```
