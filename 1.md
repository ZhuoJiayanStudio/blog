# Gitbook配置RSS以及Github加速
根据网上查到的方法，Gitbook配置RSS源只需要安装一个插件。但是，在Gitbook.io上安装插件好像要升级，我都用Gitbook.io了，我邊度嚟嘅米？但话又说回来，Gitbook.io集成Github是免费的，所以我们可以通过Github来配置RSS源，但是貌似只能看到commit的说明。
## Gitbook集成Github
具体的步骤忘记录了，但是我们可以查看Gitbook官方文档：[启用 GitHub 同步](https://gitbook.com/docs/documentation/zh/getting-started/git-sync/enabling-github-sync)。
## Github配置RSS
网上有一篇教程：[添加一个github的rss订阅](https://segmentfault.com/a/1190000040474594)。

根据教程的内容，Github有自带的RSS，格式如下：https://github.com/:owner/:repo/commits.atom 。

所以，我们只需要把用户名和仓库名写上去，就行了。我的RSS订阅源如下：https://github.com/ZhuoJiayanStudio/blog/commits.atom 。

问题来了，由于GFW，国内基本无法访问Github，怎么办呢？
## Github加速
网上有一个大佬开发了一个加速器，叫dev-sidecar，直接进入链接下载安装即可：[dev-sidecar](https://gitee.com/shun_dev/dev-sidecar)。

玄离也推荐过一个镜像站，叫xget，网址格式：https://xget.xi-xu.me/gh/你的Github内容 。
